Have the function PrimeTime(num) take the num parameter being passed and return the string true
if the parameter is a prime number,
otherwise return the string false. The range will be between 1 and 2^16.

Have the function PrimeMover(num) return the numth prime number. The range will be from 1 to 10^4.
For example: if num is 16 the output should be 53 as 53 is the 16th prime number.
